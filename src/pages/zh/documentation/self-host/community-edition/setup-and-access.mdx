# 设置和访问

了解如何设置和访问 Hoppscotch 社区版。

成功运行必要的容器后，下一步是使用管理员帐户来管理 Canyon。

import { Callout } from 'nextra/components';

<Callout type="info" emoji="ℹ️">
  系统会自动设置一个默认的管理员帐户。您可以使用以下凭据登录： username: canyon，password: 123456
</Callout>

## 管理员登录

1. 打开浏览器并访问 [http://localhost:8000/sys-settings](http://localhost:8000/sys-settings)。
2. 根据页面提示对Canyon进行初始化设置。
3. 配置完成后，您可以通过 [http://localhost:8000/login](http://localhost:8000/login) 登录到 Canyon。
