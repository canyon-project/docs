## 简而言之

Canyon是专为端到端测试而设计的。我们解决了开发人员和 QA 工程师在测试现代应用程序时面临的关键痛点。

Cypress 最常与 Selenium 进行比较；然而，Cypress 在根本上和架构上都是不同的。Cypress 不受与 Selenium 相同的限制。

这使您能够编写更快、更容易且更可靠的测试。

## 谁使用Canyon

我们的用户通常是使用现代 JavaScript 框架构建 Web 应用程序的开发人员或 QA 工程师。

Cypress 使您能够编写所有类型的测试：

- aaaa
- bbbb

Cypress 可以测试浏览器中运行的任何内容。

## Canyon生态

Cypress 包含免费、 开源、 本地安装的应用程序 和用于记录测试的Cypress Cloud 。

- 第一：当您在本地构建应用程序时，赛普拉斯每天都会帮助您设置并开始编写测试。TDD 处于最佳状态！
- 稍后：构建一套测试并将 Cypress与您的 CI 提供商集成后，Cypress Cloud可以记录您的测试运行。您永远不必想知道：为什么会失败？
