# 介绍

## 简而言之

Canyon 是一个 JavaScript 代码覆盖率解决方案。通过简单的 Babel 插件配置，即可实现覆盖率上传和实时报告生成，解决了开发人员和 QA 工程师在端到端测试中收集代码覆盖率难的问题。
